## Project Description:
                Photographer portfolio responsive and interactive landing pagee

## Highlights:
- semantic HTML
- readable and easy to maintain code
- modular JavaScript
- event handling
- event delegation
- arrow functions 
- localStorage mechanism 

## Features: 
- adaptive menu
- adaptive background image
- image caching
- changing dynamically images source in portfolio section
- multiply language translation
- switching between light and dark theme

## Techs:
 * HTML5
 * CSS, SCSS
 * Vanilla JavaScript
 * ECMAS 6
